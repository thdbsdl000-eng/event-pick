# 배너 이미지 추가 방법

이 폴더에 SNS용 이벤트 배너 이미지를 저장하세요.

## 이미지 파일 추가 방법

1. 이 폴더(`public/banners/`)에 이미지 파일을 복사하세요
2. 파일명을 다음과 같이 지정하세요:
   - `example1.jpg` (또는 `.png`, `.webp` 등)
   - `example2.jpg`
   - `example3.jpg`

## 지원하는 이미지 형식

- JPG/JPEG
- PNG
- WebP

## 권장 사항

- 이미지 크기: 가로 800px ~ 1200px 권장
- 파일 크기: 500KB 이하 권장 (빠른 로딩을 위해)
- 비율: 4:3 또는 16:9 비율이 보기 좋습니다

## 이미지가 보이지 않을 때

- 파일명이 정확한지 확인하세요 (`example1.jpg`, `example2.jpg`, `example3.jpg`)
- 파일이 `public/banners/` 폴더에 있는지 확인하세요
- 브라우저를 새로고침하세요 (Ctrl + F5)

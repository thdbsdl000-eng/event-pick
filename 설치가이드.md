# EventBanner AI 설치 가이드

## 1단계: Node.js 설치 (필수)

이 프로젝트를 실행하려면 Node.js가 필요합니다.

### 설치 방법:
1. 웹 브라우저에서 https://nodejs.org 접속
2. **LTS 버전** (추천) 다운로드 버튼 클릭
3. 다운로드된 설치 파일 실행
4. 설치 과정에서 **"Add to PATH"** 옵션이 체크되어 있는지 확인
5. 설치 완료 후 **컴퓨터 재시작** (중요!)

### 설치 확인:
설치 후 PowerShell이나 명령 프롬프트를 새로 열고 다음 명령어 입력:
```
node --version
npm --version
```

두 명령어 모두 버전 번호가 나오면 성공입니다!

---

## 2단계: 프로젝트 의존성 설치

Node.js 설치가 완료되면, 프로젝트 폴더에서 다음 명령어를 실행하세요:

```
npm install
```

이 명령어는 프로젝트에 필요한 모든 라이브러리를 다운로드합니다.
(몇 분 정도 걸릴 수 있습니다)

---

## 3단계: Google Gemini API 키 발급

1. 웹 브라우저에서 https://makersuite.google.com/app/apikey 접속
2. Google 계정으로 로그인
3. "Create API Key" 버튼 클릭
4. 생성된 API 키를 복사

---

## 4단계: 환경 변수 설정

프로젝트 폴더에 `.env.local` 파일을 생성하고 다음 내용을 입력하세요:

```
GEMINI_API_KEY=여기에_발급받은_API_키_붙여넣기
```

예시:
```
GEMINI_API_KEY=AIzaSyAbc123def456ghi789jkl
```

---

## 5단계: 개발 서버 실행

다음 명령어로 웹사이트를 실행하세요:

```
npm run dev
```

성공하면 다음과 같은 메시지가 나타납니다:
```
- ready started server on 0.0.0.0:3000
- Local: http://localhost:3000
```

웹 브라우저에서 http://localhost:3000 을 열면 사이트를 사용할 수 있습니다!

---

## 문제 해결

### npm 명령어가 인식되지 않을 때
- 컴퓨터를 재시작해보세요
- Node.js 설치 시 "Add to PATH" 옵션이 체크되었는지 확인하세요
- PowerShell이나 명령 프롬프트를 **관리자 권한**으로 실행해보세요

### API 키 오류가 발생할 때
- `.env.local` 파일이 프로젝트 루트 폴더에 있는지 확인하세요
- API 키 앞뒤에 공백이나 따옴표가 없는지 확인하세요
- 서버를 재시작해보세요 (Ctrl+C로 중지 후 다시 `npm run dev`)

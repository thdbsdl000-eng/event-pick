# 배너 이미지 추가 가이드

## 📁 이미지 저장 위치

프로젝트 폴더 안의 `public/banners/` 폴더에 이미지 파일을 저장하세요.

**전체 경로 예시:**
```
C:\Users\82109\Desktop\event pick\public\banners\
```

## 📝 파일명 규칙

다음과 같은 파일명으로 저장하세요:

- `example1.jpg` (또는 `.png`)
- `example2.jpg`
- `example3.jpg`

## 🖼️ 이미지 추가 방법

### 방법 1: 파일 탐색기 사용
1. 파일 탐색기를 열고 프로젝트 폴더로 이동
2. `public` → `banners` 폴더 열기
3. 이미지 파일을 복사해서 붙여넣기
4. 파일명을 `example1.jpg`, `example2.jpg`, `example3.jpg`로 변경

### 방법 2: 드래그 앤 드롭
1. 이미지 파일을 선택
2. `public/banners/` 폴더로 드래그 앤 드롭
3. 파일명 변경

## ✅ 확인 방법

이미지를 추가한 후:
1. 브라우저에서 `http://localhost:3000` 새로고침 (Ctrl + F5)
2. 첫 화면에 이미지가 표시되는지 확인
3. 마우스를 올려서 확대 효과가 작동하는지 확인

## 💡 팁

- 이미지가 많으면 `example4.jpg`, `example5.jpg` 등으로 계속 추가 가능
- 코드에서 `exampleBanners` 배열에 경로를 추가하면 더 많은 이미지를 표시할 수 있습니다
